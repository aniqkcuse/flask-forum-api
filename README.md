# Forum flask API Rest

This is a personal api rest that use a basic crud to: User, Topic, Discussion, Comment.

Main feature:

- Authentication works with JWT
- Testing using pytest and postman.
- Documentation generated by openapi and swagger.
- Mysql as the database.
- Gunicorn and nginx as wsgi and proxy reverse.
- Docker compose for portability

---

With docker installed, you can execute the backend system in the port 1337.

```
docker compose up -d --build
```
